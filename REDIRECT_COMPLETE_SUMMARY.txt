┌─ REDIRECT LOGIC FLOW ──────────────────────────────────────────────────────┐
│                                                                              │
│  After Successful Login:                                                    │
│                                                                              │
│  1. Check user.role                                                         │
│     ├─ If "admin" → Redirect to /admin                                     │
│     └─ If "user" → Check for callbackUrl                                   │
│        ├─ If callbackUrl exists → Redirect to callbackUrl                  │
│        └─ If no callbackUrl → Redirect to /packages                        │
│                                                                              │
│  Protected Route Access (Unauthenticated):                                  │
│                                                                              │
│  1. User tries to access protected page                                     │
│  2. useProtectedRoute() detects no auth_token                              │
│  3. Capture current URL as callbackUrl                                      │
│  4. Redirect to /login?callbackUrl=%2Fcurrent-page                         │
│  5. User logs in                                                            │
│  6. LoginForm detects callbackUrl in query string                          │
│  7. After successful auth, redirect to callbackUrl                         │
│  8. User arrives at originally requested page                              │
│                                                                              │
└────────────────────────────────────────────────────────────────────────────┘

┌─ USAGE EXAMPLES ───────────────────────────────────────────────────────────┐
│                                                                              │
│ EXAMPLE 1: Protect a Page (Auto Redirect on Login)                          │
│ ────────────────────────────────────────────────────────────────            │
│                                                                              │
│   'use client';                                                              │
│   import { useProtectedRoute } from '@/hooks/useProtectedRoute';            │
│                                                                              │
